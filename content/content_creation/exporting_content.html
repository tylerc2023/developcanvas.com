--- title: exporting content description: tags: - section topic: content creation subtopic: exporting content index: 3 --- # Exporting Assets PlayCanvas can import 3D content in three different formats: FBX, 3DS and COLLADA (DAE). We strongly recommend you use FBX since this the most robust and well tested interchange format. It also supports embedding of textures which preserve your material-texture relationships on import to PlayCanvas. Here are some tips for exporting to the FBX file format: * Ensure you have the latest available FBX exporter for your modelling application installed. * Only export what you need from your scene to FBX. For example, if your scene contains a red car and a blue car but you only want to import the red car, export only the red car to FBX. Your modelling application should have an 'Export Selected' option. * Check the 'Embed Media' option in the FBX exporter panel. * Select the 'Binary' file format option in the FBX exporter panel.
