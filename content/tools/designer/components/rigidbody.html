--- title: rigidbody component description: topic: components index: 13 --- # Component: rigidbody The 'rigidbody' component enables an entity to participate in the scene's physics simulation. This allows the movement of an entity to be simulated realistically. Note that you must add a [collision component](/tools/designer/components/collision.html) in order to define the shape of the rigid body. Otherwise, the rigidbody component has no effect and will not participate in the physics simulation. ## Properties
PropertyDescription
MassThe mass of the body. If world units in your scene are meters, units for mass is kilograms.
Linear DampingSpecifies the proportion of linear velocity is that lost by the body every second.
Angular DampingSpecifies the proportion of angular velocity is that lost by the body every second.
Linear FactorMultiplier for a body's linear movement in each world axis.
Angular FactorMultiplier for a body's angular movement in each world axis.
FrictionControls how a body loses velocity when in contact with other bodies.
RestitutionA measure of the bounciness of a body between 0 and 1.
Body TypeThe type of the body:
  • Static
  • Dynamic
  • Kinematic
## Scripting Interface You can control a rigidbody component's properties using a [script component](/tools/designer/components/script.html). The rigidbody component's scripting interface is [here](/engine/api/stable/symbols/pc.fw.RigidBodyComponent.html).
