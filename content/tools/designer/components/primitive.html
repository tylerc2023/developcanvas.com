--- title: primitive component description: topic: components index: 12 --- # Component: primitive The 'primitive' component attaches a 3D graphical primitive to an entity. ## Properties
PropertyDescription
TypeThe shape of the primitive. Can be:
  • Box
  • Capsule
  • Sphere
  • Cylinder
  • Cone
ColorThe color of the primitive.
Cast ShadowsIf enabled, the model rendered by this component will cast shadows onto other models in the scene.
Receive ShadowsIf enabled, the model rendered by this component will receive shadows cast by other models in the scene.
## Scripting Interface You can control an primitive component's properties using a [script component](/tools/designer/components/script.html). The primitive component's scripting interface is [here](/engine/api/stable/symbols/pc.fw.PrimitiveComponent.html).
