--- title: directionallight component description: topic: components index: 9 --- # Component: directionallight The 'directionallight' component attaches a directional light source to an entity. ## Properties
PropertyDescription
EnableIf checked, the directional light will light the scene.
ColorThe color of the emitted light.
IntensityThe intensity of the light that acts as a scalar value for the light's color. This value can exceed 1.
Cast ShadowsIf checked, the light will cause shadow casting models to cast shadows.
Shadow ResolutionThe resolution of the shadowmap generated by this light source. This property is only used when Cast Shadows is checked. A high value will result in a more accurate shadow but will be at the cost of performance.
## Scripting Interface You can control an directionallight component's properties using a [script component](/tools/designer/components/script.html). The directionallight component's scripting interface is [here](/engine/api/stable/symbols/pc.fw.DirectionalLightComponent.html).
