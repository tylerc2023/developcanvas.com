--- title: spotlight component description: topic: components index: 16 --- # Component: spotlight The 'spotlight' component attaches a spotlight source to an entity. ## Properties
PropertyDescription
EnableIf checked, the directional light will light the scene.
ColorThe color of the emitted light.
IntensityThe intensity of the light that acts as a scalar value for the light's color. This value can exceed 1.
Attenuation EndThe distance from the spotlight source at which its contribution falls to zero.
Inner Cone AngleThe angle from the spotlight's direction at which light begins to fall from its maximum.
Outer Cone AngleThe angle from the spotlight's direction at which light falls to zero.
Cast ShadowsIf checked, the light will cause shadow casting models to cast shadows.
Shadow ResolutionThe resolution of the shadowmap generated by this light source. This property is only used when Cast Shadows is checked. A high value will result in a more accurate shadow but will be at the cost of performance.
## Scripting Interface You can control an spotlight component's properties using a [script component](/tools/designer/components/script.html). The spotlight component's scripting interface is [here](/engine/api/stable/symbols/pc.fw.SpotLightComponent.html).
