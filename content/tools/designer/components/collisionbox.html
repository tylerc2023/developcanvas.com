--- title: collisionbox component description: topic: components index: 5 --- # Component: collisionbox The 'collisionbox' component attaches a box-shaped collision volume to an entity. ## Properties
PropertyDescription
Half ExtentsThe half-extents of the collision box. This is a 3-dimensional vector: local space half-width, half-height, and half-depth.
## Scripting Interface You can control an collisionbox component's properties using a [script component](/tools/designer/components/script.html). The collisionbox component's scripting interface is [here](/engine/api/stable/symbols/pc.fw.CollisionBoxComponent.html).
