--- title: pointlight component description: topic: components index: 11 --- # Component: pointlight The 'pointlight' component attaches a point light source to an entity. ## Properties
PropertyDescription
EnableIf checked, the point light will light the scene.
ColorThe color of the emitted light.
IntensityThe intensity of the light that acts as a scalar value for the light's color. This value can exceed 1.
RadiusThe radius of the point light (a point light source is spherical).
## Scripting Interface You can control an pointlight component's properties using a [script component](/tools/designer/components/script.html). The pointlight component's scripting interface is [here](/engine/api/stable/symbols/pc.fw.PointLightComponent.html).
