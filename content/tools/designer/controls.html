--- title: controls description: topic: playcanvas designer subtopic: controls index: 2 --- # PlayCanvas Designer Controls ## Camera Controls The Designer's Camera is controlled with the Mouse and Keyboard
ALT + Left Mouse ButtonOrbit camera
ALT + Middle Mouse ButtonPan camera
CMD + ALT + Left Mouse ButtonPan camera
Mouse WheelDolly camera
## Mouse Controls
Left Mouse ButtonSelect entity
Right Mouse ButtonActivate popup menu (in 3D view)
## Keyboard Controls
DELETEDeleteDeletes the current entity selection
SPACEToggle QuadviewToggles the quadview
DDuplicateDuplicates the selected entity hierarchy as a new sibling in the pack explorer
ERotateChange gizmo mode to rotate
FFrame SelectionFrame the active camera on the entity selection
RScaleChange gizmo mode to scale
WTranslateChange gizmo mode to translate
CTRL+ENew EntityCreates a new entity as a child of the currently selected entity
CTRL+LLaunch LocalRuns the currently loaded pack, using local server address for scripts
CTRL+SHIFT+LLaunchRuns the currently loaded pack, using scripts from the code repository
CTRL+KNew ComponentActivates the add component dialog for the currently selected entity
CTRL+ZUndoUndo the last action performed by the user
CTRL+SHIFT+ZRedoRedo the next action performed by the user
