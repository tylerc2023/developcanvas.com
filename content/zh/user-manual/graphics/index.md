PlayCanvas采用了先进的图形引擎，内部采用了WebGL API绘制图形图元。

主要有以下几个关键特征：

* Physically based rendering (PBR)
* Directional, point and spot lights (all of which can cast shadows)
* Static and skinned mesh rendering
* GPU particle engine
* Post effect library
