# 获得帮助

## 一起聊聊？

来聊聊你的游戏，展示你最近的项目，寻找新的团队伙伴或者为别人贡献你的帮助都是极好的。来玩我们的 [论坛][10] 参与讨论。

## 有问题了？

如果遇到了关于 PlayCanvas 的技术方面问题，譬如 API 或平台方面的，试着使用问答平台 [PlayCanvas Answers][11] 获取帮助。
